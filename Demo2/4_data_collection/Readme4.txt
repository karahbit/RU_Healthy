PURPOSE
This folder contains files with data collection for RU Healthy project. 

Firebase_Database_Datatree_format.pdf

Shows the Firebase JSON Tree format for data collection. 
----------------------------------------------------------------------------------------

Firebase_Database_Collection.pdf

Contains data collection from the RU Healthy Android and Web applications. 
The patient data was collected from the mobile application.  
For activity information, the data is arranged by dates in following format:
YEARMMDD where Year refers to the year, MM refers to month and DD refers to the day.
